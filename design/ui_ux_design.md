# UI / UX Design

## Target Users

- General readers
- Journalists
- Publishers

## Pages / Screens

1. Home Page – displays latest approved articles
2. Article Detail Page – displays full article
3. Journalist Dashboard – manage articles
4. Publisher Dashboard – approve or reject articles
5. Login / Registration pages

## UX Considerations

- Simple navigation menu
- Clear article categories
- Readable typography
- Mobile responsive layout
- Clear feedback messages (success / error)
